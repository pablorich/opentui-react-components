# react

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

To lint code:

```bash
bun lint
```

To auto-fix linting issues:

```bash
bun lint:fix
```

To check formatting:

```bash
bun format
```

To auto-format code:

```bash
bun format:fix
```

This project uses [Biome](https://biomejs.dev/) for linting and formatting. Pre-commit hooks are configured to run the linter automatically.

This project was created using `bun create tui`. [create-tui](https://git.new/create-tui) is the easiest way to get started with OpenTUI.
